# Frontend - Infraestructura Caja de Abogados

Aplicación web moderna construida con React, TypeScript y Vite para la gestión integral de infraestructura.

## Tecnologías
- React 18 + TypeScript
- Vite (build tool)
- TailwindCSS (estilos)
- Axios (HTTP client)
- React Router (navegación)

## Instalación
```bash
npm install
```

## Variables de entorno
Crear archivo `.env` con:
```
VITE_API_URL=http://localhost:4000/api
```

## Desarrollo
```bash
npm run dev
```

## Construcción para producción
```bash
npm run build
```

## Funcionalidades
- 🔐 Autenticación JWT
- 👥 Gestión de usuarios y roles
- 📊 Monitoreo Zabbix integrado
- 📚 Base de conocimientos (estilo Zammad)
- 🔗 Gestión de enlaces útiles
- 📱 Diseño responsive
